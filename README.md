# Bird Guard

## Installation

For users:

```pip install -r requirements.txt```

For development:

```pip install -r requirements-dev.txt```

## Cam lib

Clone repo for dev (access to examples etc.):

```
git clone ...
cd ...

python -m venv .venv
.venv\Scripts\activate

pip install -e ./libs/camera_core
```

Then run examples:

```python libs/camera_core/examples/simple_capture.py```

For api usage only (src dir):
```pip install camera-core```
